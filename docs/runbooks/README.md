# Runbooks

Operational procedures for building, installing, troubleshooting, and maintaining ECC.

## Conventions

- Each runbook covers one operational domain
- Steps are concrete actions (commands to run, values to check)
- Include expected output for verification commands
- Link to dashboards and logs where relevant
